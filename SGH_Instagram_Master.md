## 03 KPI（フォロワー数ではない）

| 指標 | 目安 | 理由 |
|---|---|---|
| 保存率 | 5%以上 | 旅行は「後で見返す」 |
| プロフィールクリック率 | 1〜2% | 「泊まりたい人の数」 |
| Airbnbリンククリック | 月30〜50 | 予約に最も近い指標 |
| 予約数 | 月3〜10 | 最終KPI |
| 指名検索数 | 増加傾向 | ブランド化の指標 |

フォロワー数は結果であって目的ではない。

### 投稿要素の重要度（売上への影響）

| 重要度 | 要素 |
|---|---|
| ★★★ | Instagram投稿の写真クオリティ |
| ★★★ | Airbnb / Booking.comのページ写真 |
| ★★ | 価格設定・レビュー数 |
| ★ | キャプション |
| ★ | 投稿頻度 |

→ 写真撮影・Airbnb写真改善が最も売上に直結する投資。次フェーズの優先タスク。

## 09 広告設計（Airbnb予約目的）

### 方針
@sgh.okinawaの投稿を広告に転用。Airbnbページへ直接誘導。

### 最初の構成（2週間テスト）

| 項目 | 内容 |
|---|---|
| 施設 | Villa JYANA（今帰仁）先行 |
| 素材 | 縁側かサウナの静止画1枚 |
| キャプション | 型O-1をそのまま使用 |
| リンク | AirbnbのVilla JYANA直リンク |
| 期間 | 2週間 |
| 予算 | 1〜3万円でテスト |

### ターゲット設定

| 項目 | 設定 |
|---|---|
| 年齢 | 28〜45歳 |
| 性別 | 全性別（女性比率高め） |
| 地域 | 東京・大阪・福岡・名古屋 |
| 興味関心 | 旅行 / 沖縄 / 一棟貸し / Airbnb / サウナ |
| 除外 | 沖縄在住者 |

### 拡張手順
2週間後にプロフリンクタップ数とAirbnbのページ流入を比較。
効果が出たら → 予算を増やす / 施設を追加（URUMA DOME → Miyako House）
Reels広告への切り替えは静止画の数値確認後。
