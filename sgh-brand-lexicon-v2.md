## SGH Resorts（親ブランド）

### 0. Core / コア

**Stay where time moves differently.**
時間の流れが、違う場所に泊まる。

### 1. Definition / 定義

SGH Resortsは、観光施設ではない。
ホテルチェーンでもない。ブランドでもない。

T2が各地で運営している、
「その土地の時間に滞在するための拠点」の総称だ。

施設の形は、一棟貸切のヴィラもあれば、
街の中の小さなホテルもある。

共通しているのは、
観光のための場所ではなく、
その土地の時間の中に入るための滞在拠点であること。

---

*English definition:*

SGH Resorts is not a tourist facility.
Not a hotel chain.

It is a collective name for the places operated by T2 in different regions.

Some are private villas.
Some are small local hotels.

What they share:
they are places to stay and enter the local time of each land.

### 2. Tone / トーン

- **低い・静かな**：声を張らない。すでにそこにいる人に話しかける温度
- **地に足がついている**：雰囲気だけで止まらない。具体的な場所・距離・特徴を言う
- **法人にも届く説得力**：哲学で飾らず、機能と体験を両立させる
- 感嘆符は使わない
- 「〜を体感」「〜な体験」の多用は避ける（陳腐化する）

### 3. 語彙 二層構造

| 内部設計語（スタッフ・設計の言葉） | 外向けコピー語（発信・プロフィールの言葉） |
|---|---|
| 一棟貸切（国内施設） | 完全貸切 / 独り占め |
| ホテル（カンボジア） | ローカルホテル / 小さなホテル |
| ローカル体験 | その土地の時間 / 地元の日常 |
| 法人・団体 | グループ / チーム / 仲間 |
| 宿泊 | 滞在 / 過ごす |
| アクセス良好 | 〜まで○分 |
| リトリート | 何もしない時間 / 山の静けさ |

※「完全貸切」は国内施設（沖縄・宮古島・箱根）のみに使う。SGH全体のコンセプト語として使わない。

### 4. Words to Avoid / 避ける語

**日本語**
| NG語 | 理由 |
|---|---|
| 贅沢な | ありふれている。何が贅沢かを具体的に言う |
| 特別な体験 | 抽象的すぎる。何が特別かを言う |
| 癒し | 観光パンフレットの語。使わない |
| 絶景 | 写真で見せる。言葉で言わない |
| 思い出づくり | ターゲットが求めていない言葉 |
| ご利用いただけます | 硬すぎる。「使える」「泊まれる」に言い換える |

**英語**
| NG語 | 理由 |
|---|---|
| luxury | 文脈なく使うと陳腐。具体的な要素を言う |
| curated | D2Cテンプレ語。使わない |
| experience | 「泊まる」「行く」「食べる」に言い換える |
| escape | 旅行広告の常套句 |
| mindful / intentional | Instagram的AI語。禁止 |
| hidden gem | 観光メディア語。禁止 |

### 5. 言い換え辞書

| NG | 代替 |
|---|---|
| 贅沢なひとときを | 自分のペースで動く数日間 |
| 特別な体験 | 〇〇ができる / 〇〇まで○分 |
| 癒しのリゾート | 何もしない時間が、ここにある |
| ご宿泊いただけます | 泊まれます / 使えます |
| escape | 日常を離れる → 削除。「その場所の時間に入る」 |
| luxury stay | private stay / one-building rental |

### 6. 文体ルール

- **主語は「場所」か「泊まる人の状態」**。物件の機能を主語にしない
  - NG：「URUMA DOMEはオーシャンビューを楽しめる施設です」
  - OK：「目の前に海。波音だけで目が覚める朝。」
- 短文を重ねる。長い一文より短い三文
- 数字を使う：「〜まで5分」「最大6名」「65㎡」
- 日本語と英語を混在させる場合、英語は固有名詞・場所名のみに絞る

### 7. 説明の型（SNS・物件紹介）

```
① 場所の状態（景色・音・空気）
② 物件の特徴（一棟・何名・何がある）
③ 周辺の具体情報（〜まで○分）
④ 誰向けか（グループ・法人・家族 など）
⑤ 予約導線（Airbnb / Booking.com）
```

## Local Time Collection

### コア

**その土地の時間に、溶け込む。**
Enter the rhythm of a place, not just visit it.

### 定義

観光地をまわる旅ではない。
地元の人が毎朝通う道に、勝手に混ざる旅だ。
コンビニへの帰り道に、夕暮れが綺麗だったりする。
Local Timeは、そういう時間のための場所を貸す。

### トーン

- **生活的・具体的**：「地元の人が〜」「朝7時の〜」のような解像度で言う
- 旅行感を強調しすぎない。「暮らすように泊まる」に近い感覚
- 沖縄なのに「南国感」を押さない。その場所の固有性を言う

### 使える短文集

```
「観光地ではなく、その土地の日常へ。」
「スケジュールを決めなくていい。」
「地元の食堂の場所は、着いてから教えます。」
「世界遺産のすぐ近くに、誰も来ない時間帯がある。」
「完全貸切。ほかのゲストを気にしなくていい。」
「その土地の時間が、旅になる。」
「Local, not tourist.」
「Stay as a base. Explore as a local.」
```

## 最終運用ルール

**「その場所でしか過ごせない時間を、具体的に言う。」**

雰囲気だけで止まらない。場所・距離・特徴・人数を言ってから、余白を残す。
